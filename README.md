# LouvorJA

## Configuração do Projeto
```
npm install
```

### Executa e recarrega automaticamente para desenvolvimento
```
npm run serve
```
ou
```
npm run dev
```

### Compila e minifica para produção
```
npm run build
```

### Visualiza o build de produção
```
npm run preview
```

### Executa o linter e corrige arquivos
```
npm run lint
```
